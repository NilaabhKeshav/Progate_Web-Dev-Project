HTML

Hypertext Markup Language, a standardized system for tagging text files to achieve font, colour, graphic, and hyperlink effects on World Wide Web pages.

CSS

Cascading Style Sheets is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

Javascript

JavaScript is a client scripting language which is used for creating web pages, and add special effects on pages like rollover, roll out and many types of graphics.

The above three tools form the basis of web development.

Comparing web development to a sentence, HTML would be the nouns, CSS the adjectives and Javascript the verbs of the sentence.

Using the above three, we can create web pages varying from simple to advanced.
